**Меблерія** — вебзастосунок, що пропонує користувачам високоякісні меблі, які
поєднують **дизайн, довговічність та доступність**.

---

## Про проєкт

Місія проєкту — надати користувачам зручну та функціональну онлайн-платформу для
перегляду, вибору та замовлення меблів із різних категорій та аксесуарів до
них.  
Завдяки інтеграції з API користувач отримує актуальну інформацію про товари, їх
характеристики, рейтинг і наявність.

**Задача, яку вирішує проєкт:**

- представити зручний каталог меблів із фільтрацією та рейтингом;
- покращити взаємодію з користувачем через інтуїтивний UI;
- надати можливість швидко переглядати, оцінювати та обирати меблі онлайн.

---

## Використані технології

- **HTML5**, **CSS3**, **JavaScript (ES6)**
- **Vite** — для швидкої збірки проєкту
- **Swagger (Furniture API)** — для отримання даних про товари
- **Бібліотеки:**
  - `accordion` — для реалізації згортання/розгортання контенту
  - `css-star-rating` — для відображення рейтингу
  - `raty-js` — для інтерактивної оцінки товарів
  - `Swiper` — для слайдерів і каруселей
  - `iziToast` — для сповіщень і повідомлень користувачу

## Як запустити проєкт локально

1. **Клонувати репозиторій:** git clone
   https://github.com/Ihor-Kotliarevskyi/js-project-group-04.git
2. **Перейти до каталогу проєкту:** cd js-project-group-04
3. **Встановити залежності:** npm install
4. **Запустити локальний сервер розробки:** npm run dev
5. Відкрити у браузері адресу, виведену у терміналі (зазвичай
   `http://localhost:5173/`).
